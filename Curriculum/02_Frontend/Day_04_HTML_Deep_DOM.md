## 🧠 Теория (Глубокое погружение)

### 1. Что такое DOM?
Когда браузер скачивает ваш `.html` файл с сервера (который напишут Ega и Hip), он не просто рисует текст на экране. Он читает код сверху вниз и строит в памяти **Дерево объектов (DOM)**.
Каждый тег (например, `<p>`) становится "веткой" этого дерева. Позже, с помощью JavaScript, мы сможем брать эти ветки, двигать их, удалять и перекрашивать. Если DOM-дерево кривое — интерфейс будет тормозить.

### 2. Семантический HTML5
Новички делают сайты, состоящие из сотен тегов `<div>` (универсальный контейнер). Это работает визуально, но это **ошибка**.
- Поисковые роботы (Google) не понимают, где на сайте главное меню, а где статья, если всё это `<div>`.
- Для этого придумали **семантические теги**: 
  - `<header>` — шапка сайта (логотип, навигация).
  - `<nav>` — блок со ссылками меню.
  - `<main>` — главный уникальный контент страницы.
  - `<footer>` — подвал сайта.

### 3. Вложенность и Блочные/Строчные элементы
HTML элементы бывают двух типов:
- **Блочные (Block):** Занимают всю ширину экрана (например, `<h1>`, `<div>`, `<p>`). Следующий элемент начнется с новой строки.
- **Строчные (Inline):** Занимают только ту ширину, которая нужна тексту (например, `<a>` для ссылок, `<span>`).

## 🛠️ Практика (Задание на сегодня)

У нас пока нет дизайна (CSS), мы строим только голый скелет.

1. Установите редактор кода **VS Code** (бесплатно).
2. Создайте файл `index.html`.
3. Напишите базовую структуру семантической страницы:
   ```html
   <!DOCTYPE html>
   <html>
   <head>
       <title>Task Manager VAB-OS</title>
   </head>
   <body>
       <header>
           <h1>VAB-OS</h1>
           <nav>
               <a href="#">Задачи</a> | <a href="#">Профиль</a>
           </nav>
       </header>
       <main>
           <h2>Мои задачи на сегодня</h2>
           <ul>
               <li>Настроить Git</li>
               <li>Прочитать про DOM</li>
           </ul>
       </main>
   </body>
   </html>
   ```
4. Откройте этот файл в браузере. Выглядит некрасиво? Да. Но поисковые роботы и слепые люди (через программы чтения экрана) прочитают его идеально.
