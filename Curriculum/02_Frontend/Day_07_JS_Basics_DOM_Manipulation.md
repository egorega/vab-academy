## 🧠 Теория (Глубокое погружение)

### 1. Подключение и База JS
JS подключается в самом низу вашего HTML (перед закрывающим `</body>`), чтобы он не тормозил загрузку визуальной части сайта.
```html
<script src="script.js"></script>
```
В JS мы объявляем переменные через `let` (изменяемые) или `const` (неизменяемые). **Забудьте про `var`**, это пережиток прошлого.
```javascript
const username = "Par"; // Константа, нельзя изменить
let age = 16;           // Переменная, можно изменить
age = 17;               // Ок!
```

### 2. Что такое DOM?
Браузер читает ваш HTML-файл и превращает его в большое дерево объектов в оперативной памяти. Это дерево называется **DOM**.
JS может путешествовать по этому дереву, находить нужные теги и изменять их прямо на лету (менять цвета, удалять элементы, подставлять текст).

### 3. Поиск элементов и События (Events)
Чтобы JS что-то сделал с кнопкой, он должен её найти. Мы используем `document.querySelector()`.
```javascript
// Ищем кнопку по CSS-классу
const myButton = document.querySelector('.btn-submit');

// Вешаем "слушателя событий" (EventListener). 
// Когда кто-то кликнет ('click') на кнопку, выполнится функция.
myButton.addEventListener('click', function() {
    alert("Кнопка нажата!"); // Всплывающее окно
});
```
