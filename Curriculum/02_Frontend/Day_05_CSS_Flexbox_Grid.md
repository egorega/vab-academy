## 🧠 Теория (Глубокое погружение)

### 1. Box Model (Коробочная модель)
Всё в вебе — это прямоугольники. Абсолютно всё. Даже круглая кнопка — это прямоугольник с закругленными углами (`border-radius`).
У каждого прямоугольника есть:
- `Content` (Сам текст или картинка).
- `Padding` (Внутренний отступ от контента до рамки).
- `Border` (Сама рамка).
- `Margin` (Внешний отступ, расталкивающий соседей).

### 2. Flexbox (Гибкая коробка)
Вы создаете блок `<div>` и говорите ему: `display: flex;`. 
Всё! Теперь все его дочерние элементы становятся "резиновыми". Вы можете выровнять их по центру одной строчкой:
- `justify-content: center;` (По горизонтали)
- `align-items: center;` (По вертикали)
Flexbox идеален для навигационных меню, выравнивания иконок и кнопок. Он работает в **одном измерении** (строка ИЛИ колонка).

### 3. CSS Grid (Жесткая сетка)
Когда вам нужно построить целый дашборд (где слева меню, сверху шапка, а по центру контент), Flexbox не справится. Нужен `display: grid;`.
Grid делит экран на колонки и строки.
```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; /* Левое меню 200px, контент - все остальное место (1fr) */
}
```

## 🛠️ Практика (Задание на сегодня)

1. В папке с вашим `index.html` (из Дня 4) создайте файл `style.css`.
2. Подключите его в HTML внутри тега `<head>`: `<link rel="stylesheet" href="style.css">`
3. Сделайте `<body>` Flex-контейнером:
   ```css
   body {
       display: flex;
       flex-direction: column;
       min-height: 100vh; /* Высота на весь экран */
       margin: 0;
   }
   ```
4. Используя CSS Grid, создайте внутри `<main>` сетку из трех колонок для ваших "Задач". Поместите задачи в карточки.
5. Настройте `gap: 20px`, чтобы между карточками был красивый отступ.
