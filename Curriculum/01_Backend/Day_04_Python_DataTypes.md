## 🧠 Теория (Глубокое погружение)

### 1. Переменные — это ярлыки, а не коробки
В школе нам часто говорят, что переменная `x = 10` — это коробка с именем `x`, в которой лежит число `10`. В Python это **не так**. 
В Python число `10` создается в оперативной памяти (RAM) как независимый объект. А `x` — это просто бумажный ярлычок (ссылка), который привязан к этому объекту.
Если вы напишете `y = x`, вы не скопируете объект `10`. Вы просто привяжете еще один ярлычок `y` к тому же самому куску памяти!

### 2. Mutable (Изменяемые) vs Immutable (Неизменяемые)
Это концепция, из-за которой падают 90% серверов новичков.
- **Immutable (Строки, Числа):** Если вы создали строку `name = "Ega"`, вы **не можете** её изменить. Вы можете только создать новую строку и перевесить ярлычок.
- **Mutable (Списки - Lists, Словари - Dicts):** Это "резиновые" контейнеры. Вы можете добавлять туда новые элементы.

**Почему это важно?**
Если вы передадите Список в функцию и измените его там, он изменится во всей программе! Потому что ярлычок вел на один и тот же объект в памяти.

### 3. Строки (Strings) и Индексы
Строка — это просто массив букв. Компьютеры начинают считать с **нуля**. 
В слове `"PYTHON"`, буква `P` имеет индекс `0`, а `N` — индекс `5`.
Python позволяет брать срезы (slices): `"PYTHON"[0:2]` вернет `"PY"`.

1. Зайдите на `python.org` и установите Python (обязательно поставьте галочку "Add Python to PATH" при установке).
2. Откройте Терминал и напишите `python`. Вы попадете в интерактивную консоль Python (появятся `>>>`).
3. Напишите код:
   ```python
   a = [1, 2, 3] # Создали список
   b = a         # Привязали ярлык b к тому же списку
   b.append(4)   # Добавили число 4 через ярлык b
   print(a)      # Угадайте, что выведется?
   ```
4. Выполните код. Вы увидите `[1, 2, 3, 4]`. Вы изменили `b`, но изменилось и `a`, потому что это один объект в памяти!
